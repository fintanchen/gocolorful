# gocolorful

_A tiny colorful output library_

This repository **don't** wrapper any function but provide definitions for user.
It means that you can easy integration with other projects.

## Color Definitions

ANSI-SEQUENCE map:

| color  |    code    |
| :----: | :--------: |
| Purple | \u001b[95m |
|  Blue  | \u001b[94m |
| Yellow | \u001b[93m |
| Green  | \u001b[92m |
|  Red   | \u001b[91m |
